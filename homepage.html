Thoughts on Web Accessiblity:

Web Accessiblity is the practice of making a website accessible for people who need to surf the Web using alternative methods.

These methods could range from:

  • Captions on a video
  • Text-to-Speech and Speech-to-Speech
  • Screen Readers/Magnifiers
  • High Contrast Elements

Web Accessiblity has been standardized using the World Wide Web Consortium (W3C). This is a collaborative community which aims to develop standards for all aspects of web development. Standards like this are important to ensure that everyone has an efficient and smooth experience when browsing the web.
